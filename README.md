# ATM Machine - Java Console Application

A simple console-based ATM simulation built in Java, demonstrating OOP concepts, exception handling, and control flow.

## Features
- PIN verification (3 attempts)
- Check balance
- Deposit money
- Withdraw money
- Mini statement (last 5 transactions)

## Tech Stack
- Java
- Scanner (user input)
- ArrayList (transaction history)
- Exception Handling (try-catch)

## How to Run
```bash
javac ATM.java
java ATM
```

## Concepts Used
- Loops (while)
- Conditionals (if-else)
- Exception handling
- Collections (ArrayList)
